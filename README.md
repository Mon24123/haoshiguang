# 好时光 Good Time

好时光是一款桌面常驻时间记录工具。它把“现在正在做什么”和“已经做了多久”放在桌面上，让记录时间不再需要先打开一堆软件、表格或文档。

这个项目最初来自一个很具体的需求：我想知道一天的时间到底花到哪里去了，但又不想被复杂的项目管理工具打断。好时光因此选择了一个更轻的方向：先把当前任务记录下来，再在一天结束时复盘。

## 为什么做它

很多时间记录工具的问题不是“不够强”，而是“开始记录太麻烦”。用户真正需要记录的时刻，往往正好是刚开始工作、学习、写东西、处理生活事务的时候。如果这时还要打开网页、进入项目、找到分类、填表格，记录本身就变成了新的负担。

好时光试图解决的是这个前置痛点：

- 当前任务始终可见，减少忘记自己在做什么的情况。
- 计时入口足够轻，不用先整理完整系统再开始。
- 允许后期修正，符合真实生活里“忘记点结束”“分类点错了”“中途切换任务”的混乱状态。
- 导出内容能进入飞书、Obsidian、表格等工具，服务于复盘而不是锁在应用里。

## 核心功能

### 桌面悬浮卡片

- 桌面常驻显示当前事项、分类和计时。
- 支持开始、暂停、结束、快速备注。
- 支持收起成屏幕边缘小胶囊，降低占屏干扰。
- 支持任务快速切换，适合频繁切换工作内容的人。

### 今日任务

- 添加当天要做的任务。
- 勾选完成后会同步停止当前计时。
- 支持从任务列表直接开始或继续计时。
- 支持删除确认，避免误删。

### 待办池

- 收纳“以后要做，但还不想排到今天”的事项。
- 支持按分类管理。
- 可以从待办池直接开始计时。

### 每日固定任务

- 适合每天重复出现的任务。
- 支持名称、分类、固定时间、预计时长。
- 可以在需要时展开查看和修改。

### 时间记录与复盘

- 同一任务可以多次继续，形成多段时间记录。
- 支持补记事项、分类和分钟数。
- 支持修改时间段、分类和备注。
- 支持复制日报。
- 支持按日、周、月复制分类明细表格，便于粘贴到飞书表格、Excel 或其他复盘工具。
- 支持将分类明细一键导出为 CSV 文件，方便后续做数据复盘。

### 数据备份

- 支持导出本地 JSON 备份。
- 数据默认保存在本机 Electron userData 目录。
- 当前没有账号系统、云同步或自动上传。

## AI 协作开发说明

好时光是一个 AI 协作完成度较高的个人项目。我主要负责需求判断、产品反馈、交互取舍和内测验证，使用 Codex / OpenClaw 等 AI Agent 协助完成代码实现、问题定位、界面调整和打包发布。

开发过程不是一次性生成，而是围绕真实使用持续迭代：

- 根据截图反馈定位 UI 错位、悬浮窗遮挡、点击状态不同步等问题。
- 根据内测体验调整任务切换、退出确认、分类编辑、备份和导出。
- 通过交接文档维护上下文，让新对话也能接着项目继续开发。
- 使用 GitHub 管理版本，方便回退、对比和公开展示。

这个项目的重点不只是“用了 AI 写代码”，而是把 AI 当成长期协作的开发伙伴，在真实需求、真实反馈和真实迭代里完成一个可用产品。

## 本地运行

安装依赖：

```bash
npm install
```

启动开发版：

```bash
npm start
```

打包 macOS：

```bash
npm run package:mac
```

打包 Windows：

```bash
npm run package:win
```

## 当前边界

- 当前没有自动更新，新版本需要重新下载或重新打包。
