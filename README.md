# 好时光

好时光是一个桌面常驻时间记录挂件，用来快速记录当前正在做的事、花了多久、属于哪个分类。

## 快速开始

安装依赖：

```bash
npm install
```

本地运行：

```bash
npm start
```

打包 macOS：

```bash
npm run package:mac
```

打包 Windows：

```bash
npm run package:win
```

## 主要入口

- 完整面板：管理任务、待办、分类统计、时间记录。
- 桌面卡片：常驻显示当前事项和计时。
- 左侧圆点：展开暂停、结束、快速备注。
- 右侧两点：展开快速切换任务。
- 右上角 `×`：收起成屏幕边缘小胶囊。
- “退出好时光”：真正关闭应用。

## 当前功能

- 开始/暂停/结束计时。
- 桌面卡片实时显示当前任务。
- 今日任务增删改、勾选完成。
- 同一任务可分多段继续记录。
- 待办池支持不限定日期的任务。
- 每日固定任务支持名称、分类、固定时间、预计时长。
- 分类支持新增、排序、改名、删除。
- 时间记录支持修改时间、备注、删除。
- 复制日报。
- macOS 和 Windows 打包。

## 数据说明

数据保存在本地 Electron userData 目录中，目前没有云同步。重新打包应用通常不会清空用户数据，但删除应用数据目录会丢失记录。

## Windows 内测说明

打包后文件夹在：

```text
dist/好时光-win32-x64
```

可发送压缩包：

```text
dist/好时光-win32-x64-内测.zip
```

Windows 用户解压后双击 `好时光.exe` 即可打开。不要只单独移动 `.exe`，需要保留整个文件夹。

## 已知限制

- 浏览器或其他应用进入系统级全屏时，桌面卡片可能被系统空间遮住，但计时仍在后台继续。
- 目前没有自动更新，新版本需要重新发安装包。
- 当前没有账号系统和云同步。
