# 好时光内测核心说明

## 这个版本能做什么

好时光是一个桌面时间记录小挂件。它会一直停在桌面上，方便你随手记录“现在在做什么”、做了多久、属于哪个分类。

## 桌面卡片怎么用

- 点卡片主体：打开/收起完整面板。
- 点左边圆点：展开暂停、结束、快速记备忘。
- 点右边两个点：展开快速切换任务。
- 点右上角 `×`：不是退出，而是收成屏幕边缘的小胶囊；点小胶囊可以恢复。
- 在展开面板里点“退出好时光”：真正关闭应用。

## 任务和时间记录

- “今日任务”适合今天明确要做的事。
- “待办池”适合不限定哪天完成的事，可以设置启动日期，也可以直接现在开始。
- 同一个任务可以分多段做，时间记录会合并展示在同一个任务下面。
- 鼠标放到某一段时间上，可以改时间、写备注、删除记录。
- 删除时间记录前会弹确认，避免误删。

## 分类

- 默认会带几个分类，例如工作、项目、生活、健康、其他。
- 分类可以自己新增、拖动排序、右键改名或删除。
- 任务、待办、固定每日任务、时间记录旁边的分类都可以改。

## 每日固定任务

- 点“查看/修改每日任务”可以维护每天自动出现的固定任务。
- 固定任务支持名称、分类、固定时间、预计分钟数。
- 以后每天打开时，会自动生成当天还没有出现过的固定任务。

## 复制日报

- “复制日报”会复制当天的任务、分类时间和时间记录。
- 复制成功后会提示“复制成功”，可以直接粘贴到聊天、文档或飞书里。

## Windows 用户怎么关掉

- 如果只是不想看到桌面卡片，点卡片右上角 `×`，它会收成边缘小胶囊。
- 如果要彻底退出，先展开卡片，再点“退出好时光”。
- 也可以在任务栏/托盘或任务管理器里结束应用。

## Windows 用户怎么放到桌面

- 解压后进入文件夹，双击 `.exe` 打开。
- 可以右键 `.exe`，选择“发送到 > 桌面快捷方式”。
- 不建议只把 `.exe` 单独拖走，因为旁边的资源文件也需要保留。

## 已知限制

- 浏览器进入系统级全屏时，桌面挂件可能会被系统放到另一个空间；计时仍会在后台继续。
- 现在还不是自动更新版本，朋友拿到新版仍需要重新下载或替换安装包。
